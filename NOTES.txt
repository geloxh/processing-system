-- Account Security
echo password_hash('yourpassword', PASSWORD_BCRYPT, ['cost' => 12]);

password_hash - This is a built-in PHP function that:
Takes a plain-text password
Returns a secure hashed version of it

BCRYPT is a strong, industry-standard hashing algorithm

cost controls how computationally expensive the hashing is
Higher cost = more secure but slower

Typical values:

10 = default (balanced)
12 = more secure (slower, but still practical)
14+ = very slow (usually unnecessary)

Trust the self-signed cert in your browser
Since XAMPP uses a self-signed cert, any browser will show a warning on first visit to https://localhost.
Chrome/Edge: click "Advanced" → "Proceed to localhost"
To permanently trust it: open C:\xampp\apache\conf\ssl.crt\server.crt → double-click → "Install Certificate" → "Local Machine" → "Trusted Root Certification Authorities"

CSP header in public/index.php
header("Content-Security-Policy: default-src 'self'; script-src 'self'; style-src 'self'; font-src 'self'; upgrade-insecure-requests;");

Production note
On a real server (cPanel, VPS, etc.), replace the self-signed cert with a free Let's Encrypt cert. The session.cookie_secure line stays exactly the same — no code change needed when you go live.
